To install INK, follow these simple steps:

1:  Download the latest ENB binaries for Fallout 4 from ENBDev.com

2:  Copy ONLY d3d11.dll and d3dcompiler_46e.dll from the ENB zip file into the root Fallout 4 game directory

3:  Copy the contents of the "Main Files" folder into the root Fallout 4 game directory
