SNAPDRAGON PRIME ENB (0.366+ BINARIES REQUIRED)
V5

March 29th, 2019

BY DEFAULT, THIS ENB IS MEANT TO BE USED FOR (CHARACTER-)SCREENSHOTS, NOT FOR GAMEPLAY.



Snapdragon Prime is (hopefully) the last big update this ENB setup will get.
The main goal hasn't changed: This is primarily meant to be used for (character) screenshots, therefore the hardware requirementsare quite high by default. I realized that quite a few people were using this ENB for gameplay, so I included optional performance settings as well as an alternative (less distracting) DoF that will boost your fps by a good amount. This is for convenience, however the settings can of course be tweaked to get even more fps - if you have some basic ENB knowledge.

This ENB also features a unique set of profiles (13+3) you can swap on-the-fly.
Check the provided help folder on how to change them ingame!

If you're looking for an ultra-realstic ENB,
this is probably not the right thing for you. The purpose of these picture profiles is to create a unique, cinematic look.
That often means blown out highlights, pitchblack darkness, and of course... a prominent Depth of Field effect.

Set your ingame brightness to the lower third, that's what I do. 30% should be about right.
Don't forget to change the VRAM value in enblocal.ini accordingly. If you don't know how, Google is your friend.
Also, open up enbbloom.fx.ini and replace the resolution with your own (default is 1920x1080).
