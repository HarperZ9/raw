Installation Steps

    Part1. Install BaseMod.oiv with OpenIV, when asked select mods folder.

    Part2. Install QuantV_Install.oiv with OpenIV (mods folder).

    Optionals: Once the main mod is installed (Part1+Part2) you may install any optional preset.

    *Remember to disable Battleye if you didn't yet. (or use the command -nobattleye)





    Trainer keycodes for QuantV.ini  (requires ScriptHookV).

    Example:
    · F5 key uses keycode 116, so in section: [MENU_KEYBOARD] is: Menu=116
    · To disable any key set it to 0
