
Uninstall any incompatible and redundant mods, Uninstall any ENB or ReShades presets.
Make sure to DELETE ENB CACHE FOLDER before using this ENB preset !
Disable from the game's launcher options: "Screen Space Ambient Occlusion" and "Antialiasing" and make sure to have the"Godrays quality" set to High.



Download, Install and enable at the very last position of your modlist the NAT-ENB plugin, let it win any conflict, load if after everything.﻿
     
Download & Install ENB Helper SE﻿ ( requires SKSE64 ) as well as EVLAS﻿.
     
Download latest ENB binaries from enbdev.com﻿﻿﻿﻿ (currently v0.493) and extract ONLY: "d3d11.dll" and "d3dcompiler_46e.dll" to the game roots folder, next to the game's .exe.
     
Download and extract the content from the preset folder/0- INSTALL MAIN FILES FIRST to the root folder of the game, next to the game's exe and ENB dlls. Accept to overwrite the EVLAS.cfg file. 
Then just select the quality version you want to use and drop the enbseries.ini file into the root folder of the game, next to the game's exe and that's it, you are done. 

The quality versions use different SSAO types preset at various quality as well as disabled or modified ENB effects settings.

The real deal is the High version, which has been used to build this mod, but feel free to use the one you prefer or that runs better for your computer.

The LOW QUALITY - BEST PERFORMANCE is most likely the best bang for the buck you can get on Skyrim SE as it doesn't rely on intensive effects while keeping clean lighting and atmoshperic settings, perfect if you want super high FPS gameplay without too much compromises on the graphics.
